## Project layout

This repository contains the following Go apps.

### 1) Go Bank
A simple banking demo.

**How to run**
```bash
cd Bank-App && go run .
```

### 2) Investment Calculator
A tool to compute the future value of investments.

**How to run**
```bash
cd investment-calc && go run .
```

### 3) Profit Calculator
A tool to calculate profit and margins.

**How to run**
```bash
cd profit-calc && go run .
```
